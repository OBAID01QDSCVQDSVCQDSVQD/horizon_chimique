# 📋 مخطط نظام التقييم والتعليقات والمراجعة

هذا المخطط يغطي طلبك لإضافة التقييمات (للحرفيين)، التعليقات والإعجابات (للمنشورات/المشاريع)، مع نظام مراجعة للأدمن.

## 1. قواعد البيانات (Database Models) 🗄️

### أ. موديل التقييم (`Review.js`)
يختص بتقييم الحرفيين (Artisans).
- **الحقول**:
  - `artisan`: (ObjectId -> User) _الحرفي المستهدف_
  - `user`: (ObjectId -> User) _صاحب التقييم_
  - `rating`: (Number 1-5) _النجوم_
  - `comment`: (String) _الملاحظة_
  - `status`: (Enum: `pending`, `approved`, `rejected`) _الحالة_
  - `timestamps`: _التاريخ_

### ب. موديل التعليق (`Comment.js`)
يختص بالتعليق على المنشورات (Chantiers/Posts).
- **الحقول**:
  - `chantier`: (ObjectId -> Chantier) _المشروع المستهدف_
  - `user`: (ObjectId -> User) _صاحب التعليق_
  - `content`: (String) _نص التعليق_
  - `status`: (Enum: `pending`, `approved`, `rejected`) _الحالة_ (افتراضياً `pending`)
  - `timestamps`: _التاريخ_

### ج. الإعجابات (`Likes`)
تُضاف مباشرة في موديل `Chantier`.
- **التعديل**: إضافة حقل `likes: [{ type: ObjectId, ref: 'User' }]` لحساب العدد ومعرفة من قام بالإعجاب.

---

## 2. واجهة المستخدم (User Interface) 💻

### أ. صفحة الحرفي (Profile)
- عرض **متوسط النجوم** (Average Rating).
- زر **"أضف تقييم"**: يفتح نافذة لكتابة الرأي وتحديد النجوم.
- رسالة بعد الإرسال: *"شكراً! تقييمك قيد المراجعة وسيظهر قريباً."*

### ب. صفحة المنشور (Post/Chantier)
- **زر الإعجاب (❤️ Like)**: تفاعلي (Live)، يلون بالأحمر عند الضغط.
- **قسم التعليقات**:
  - عرض التعليقات المعتمدة فقط (`status: approved`).
  - نموذج إضافة تعليق.
  - رسالة بعد الإرسال: *"تعليقك بانتظار موافقة الإدارة."*

---

## 3. لوحة الإدارة (Admin Panel) 🛡️

### صفحة "المراجعة" (`/admin/moderation`)
صفحة مركزية جديدة تحتوي على قسمين (Tabs):

1.  **التعليقات المعلقة (Pending Comments)**:
    - جدول يعرض: المحتوى | الكاتب | المنشور | التاريخ.
    - إجراءات: ✅ **نشر** (Approve) | ❌ **حذف** (Reject).

2.  **التقييمات المعلقة (Pending Reviews)**:
    - جدول يعرض: النجوم | التعليق | الحرفي | الكاتب.
    - إجراءات: ✅ **نشر** | ❌ **حذف**.

---

## 4. مراحل التنفيذ (Roadmap) 🚀

1.  **المرحلة الأولى**: إنشاء الموديلات (`Review`, `Comment`) وتحديث `Chantier`.
2.  **المرحلة الثانية**: برمجة الـ API (`POST` للحفظ، `GET` للعرض، `PUT` للموافقة).
3.  **المرحلة الثالثة**: إضافة مكونات الواجهة (النجوم، اللايك، التعليقات).
4.  **المرحلة الرابعة**: بناء صفحة الإدارة للمراجعة.
